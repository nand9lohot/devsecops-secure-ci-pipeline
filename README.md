# DevSecOps Secure CI Pipeline

This repository demonstrates a **secure DevSecOps CI/CD pipeline** implementing
shift-left security practices using open source tools.

## Security Controls

SAST → Semgrep  
Secrets detection → Gitleaks  
Dependency scanning → OWASP Dependency Check  
Container scanning → Trivy  
SBOM → Syft  
Vulnerability analysis → Grype  
Image signing → Cosign

## Architecture

Application Code  
↓  
SAST  
↓  
Secrets Scan  
↓  
Dependency Scan  
↓  
Container Build  
↓  
Image Scan  
↓  
SBOM Generation  
↓  
Image Signing  
↓  
Kubernetes Deployment

## Run Locally

```bash
make all
```

## Demo Results

Example outputs generated by the pipeline:

### SAST Findings
Semgrep detects command injection vulnerability.

### Secrets Detection
Gitleaks identifies leaked credentials.

### Dependency Vulnerabilities
OWASP Dependency Check identifies vulnerable dependencies.

### Container Security
Trivy detects CVEs in container images.

### SBOM
Syft generates a software bill of materials.

### Image Signing
Cosign signs container images for supply chain integrity.
